# /### Moviebot

###### Overview

The provided Python script, Movies.py, defines a command-line interface (CLI) chatbot called BollywoodMotivationBot. Its primary purpose is to recommend motivational Bollywood movies (mostly post-2017) based on specific themes or keywords provided by the user.

###### Features

Categorized movies:

Struggle to Success

Career & Ambition

Sports Spirit

Life Lessons

Displays:

Movie details (year, rating, story)

Key motivational lessons

Iconic scenes

Inspirational quotes

Keyword-based matching using regex

Randomized recommendations for variety

Interactive command-line chatbot

###### Technologies Used

Python 3

Built-in libraries:

	re (Regular Expressions)

	random

	datetime

	typing

###### Installation and Running the Project


Make sure Python is installed (Python 3+ recommended)

Run the script:
	python Movies.py

Start chatting:
	You: motivational movies about failure
	You: sports movies
	You: women empowerment films

Exit anytime by typing:
	exit / quit / bye / thanks
